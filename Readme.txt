Readme
======

The supporting files are related to the study "A method for determining the performance of formulas for calculating the crack width and spacing in reinforced concrete elements".

The following files are relevant:
Research/Method_comparing_CWFs/Data_processing_for_GitHuB_Final (contains the data processing of the database "Crack width and crack spacing in reinforced and prestressed concrete elements: database")
Research/Method_comparing_CWFs/CWF_analysis_for_GitHuB_Final (contains the evaluation of the different crack width formulas)
